== Description ==

Helps easy add Bazo.io tracker to your site

= Support =

We provide support for our plugin on [bazo.io](https://bazo.io)

* **Our Website** - https://bazo.io/

== Installation ==
* Upload the Bazo plugin to your site then Activate it.
* Enter Bazo tracker in Bazo settings page and click submit button.
* Done!
